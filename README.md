# Robert Bai — Portfolio

Static site: HTML, CSS, JS. Works as a flat file or behind any static host.

## Run

Open `index.html` in a browser, or:

```bash
npx serve .
```

## Structure

- `index.html` — layout and content
- `assets/css/style.css` — styles
- `assets/js/script.js` — nav, sidebar, portfolio filters
- `assets/images/` — avatars and project thumbnails
